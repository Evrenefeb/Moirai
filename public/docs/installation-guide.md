# Installation Guide

Project: Moirai  <br> Version: 1\.1 <br> Author\(s\): Evren Efe BALCI<br> Date: 23\.02\.2026

## 1\. Introduction

### 1\.1 Purpose

This document defines how to install system to your machine\.

### 1\.2 Target Users

- Developers
- Individuals who want to test the system

## 2\. Pre\-Requirements

PRQ\-01: Machine should have VSCode installed\.

PRQ\-02: Machine should have Git or Github Desktop installed\.

PRQ\-03: Machine should have Node\.js installed\.

PRQ\-04: Machine should have React\.js and Vite installed\.

PRQ\-05: Developer must be familiar with the technology or applications given in PRQ\-01,02,03,04\.

PRQ\-06: Developer must have an Open\-Router account and an API key\.

## 3\. Step\-by\-Step Guide

### Step 1: Clonning Repository

Clone this repository using &quot;git clone&quot; or by downloading\.

### Step 2: Shifting to IDE

After cloning this repository to your dedicated path, open it using VSCode or any external IDE of your choice\.

### Step 3: Establishing \.env

Proceed to your &quot;moirai&#x2F;openrouter\-backend&quot; folder\. Then create a file named &quot;\.env&quot;\. 

*Warning:* This folder will have your sensitive credentials\. Proceed with caution\.

### Step 4: Finishing up with \.env

Navigate towards the &quot;public&#x2F;templetes&#x2F;env\-templete\.txt&quot; and copy the content your &quot;\.env&quot; file that is mention in previous step\. Then fill the required parts with your credentials\.

### Step 5: Installing Necessary Packages

Navigate towards &quot;moirai&#x2F;openrouter\-backend&quot; and &quot;moirai&quot; folders via your terminal and use &quot;npm install&quot; to install necessary packages\.

### Step 6: Booting up

Boot up the system using &quot;npm start&quot; or &quot;npm run dev&quot;\.
